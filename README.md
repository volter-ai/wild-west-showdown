# Wild West Showdown

    A multiplayer game set in the wild west where players compete in a showdown to be the last cowboy standing.

    ## Game Features

    - Players control cowboys in a wild west setting
    - Players can move around the map, take cover, and shoot at other players
    - Power-ups like better guns, health, and speed boosts spawn on the map
    - The play area gradually shrinks, forcing players to confront each other
    - Last player standing wins the round
    - Customizable cowboy characters with different hats
    - Wild west themed environment

    ## How to Play

    1. Enter your name and create or join a game
    2. Share the game code with friends to let them join
    3. When everyone is ready, the host can start the game
    4. Use the joystick to move your cowboy
    5. Tap anywhere on the screen to shoot in that direction
    6. Use the shield button to take cover (reduces damage but slows movement)
    7. Collect power-ups to gain advantages
    8. Stay within the safe zone as it shrinks
    9. Be the last cowboy standing to win!

    ## Controls

    - Joystick (bottom right): Move your character
    - Tap screen: Shoot in that direction
    - Shield button (bottom left): Take cover/stand up

    ## Power-ups

    - ❤️ Health: Restores health
    - 👢 Speed: Increases movement speed
    - 🔫 Gun: Improves weapon (faster shooting, more damage)

    Enjoy the showdown!
